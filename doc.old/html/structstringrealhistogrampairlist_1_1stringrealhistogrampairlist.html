Detailed Description

The StringRealHistogramPairList class is a template class that provides linked lists. It stores a list of values and provides iterator-based access as well as insertions and removals. StringRealHistogramPairList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

Here's an example of a List that stores integers ...

#define StringRealHistogramPairList IntegerList
#define StringRealHistogramPairListIterator IntegerListIterator
#define class(StringRealHistogramPair) integer
#define type(StringRealHistogramPair) integer
#include "List.h90"
#undef StringRealHistogramPairList
#undef StringRealHistogramPairListIterator
#undef class(StringRealHistogramPair)
#undef type(StringRealHistogramPair)

{.c}

... and a StringRealHistogramPairList that stores Atom values:

#define StringRealHistogramPairList AtomList
#define StringRealHistogramPairListIterator AtomListIterator
#define class(StringRealHistogramPair) class(Atom)
#define type(StringRealHistogramPair) type(Atom)
#include "List.h90"
#undef StringRealHistogramPairList
#undef StringRealHistogramPairListIterator
#undef class(StringRealHistogramPair)
#undef type(StringRealHistogramPair)

{.c}

Finally, you can use the first one thus:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
call mylist.append( 8 )
call mylist.append( 5 )
call mylist.append( 1 )
iter => mylist.begin
do while ( associated(iter) )
write(*,*) iter.data
iter => iter.next
end do
