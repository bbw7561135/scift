Detailed Description

The RealList class is a template class that provides linked lists. It stores a list of values and provides iterator-based access as well as insertions and removals. RealList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

Here's an example of a List that stores integers ...

#define RealList IntegerList
#define RealListIterator IntegerListIterator
#define real(8) integer
#define real(8) integer
#include "List.h90"
#undef RealList
#undef RealListIterator
#undef real(8)
#undef real(8)

{.c}

... and a RealList that stores Atom values:

#define RealList AtomList
#define RealListIterator AtomListIterator
#define real(8) class(Atom)
#define real(8) type(Atom)
#include "List.h90"
#undef RealList
#undef RealListIterator
#undef real(8)
#undef real(8)

{.c}

Finally, you can use the first one thus:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
call mylist.append( 8 )
call mylist.append( 5 )
call mylist.append( 1 )
iter => mylist.begin
do while ( associated(iter) )
write(*,*) iter.data
iter => iter.next
end do

{.f90}
