Detailed Description

The StringIntegerPairList class is a template class that provides linked lists. It stores a list of values and provides iterator-based access as well as insertions and removals. StringIntegerPairList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

Here's an example of a List that stores integers ...

#define StringIntegerPairList IntegerList
#define StringIntegerPairListIterator IntegerListIterator
#define class(StringIntegerPair) integer
#define type(StringIntegerPair) integer
#include "List.h90"
#undef StringIntegerPairList
#undef StringIntegerPairListIterator
#undef class(StringIntegerPair)
#undef type(StringIntegerPair)

{.c}

... and a StringIntegerPairList that stores Atom values:

#define StringIntegerPairList AtomList
#define StringIntegerPairListIterator AtomListIterator
#define class(StringIntegerPair) class(Atom)
#define type(StringIntegerPair) type(Atom)
#include "List.h90"
#undef StringIntegerPairList
#undef StringIntegerPairListIterator
#undef class(StringIntegerPair)
#undef type(StringIntegerPair)

{.c}

Finally, you can use the first one thus:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
call mylist.append( 8 )
call mylist.append( 5 )
call mylist.append( 1 )
iter => mylist.begin
do while ( associated(iter) )
write(*,*) iter.data
iter => iter.next
end do

{.f90}
