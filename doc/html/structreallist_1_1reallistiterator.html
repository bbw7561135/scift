Detailed Description

This class use the List template declared into List.h90 file, please take a look to this file for details.

The RealListIterator class provides an STL-style iterator for RealList. RealListIterator allows you to iterate over a RealList and to modify the list item associated with the iterator. RealList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

You must initialize it using a function like RealList.begin before you can start iterating. Here's a typical loop that prints all the items stored in a list:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
call mylist.append( 8 )
call mylist.append( 5 )
call mylist.append( 1 )
iter => mylist.begin
do while ( associated(iter) )
write(*,*) iter.data
iter => iter.next
end do
