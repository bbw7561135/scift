The StringIntegerPairListIterator class provides an STL-style iterator for StringIntegerPairList. StringIntegerPairListIterator allows you to iterate over a StringIntegerPairList and to modify the list item associated with the iterator. StringIntegerPairList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

You must initialize it using a function like StringIntegerPairList.begin before you can start iterating. Here's a typical loop that prints all the items stored in a list:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
