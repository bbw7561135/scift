Detailed Description

The IntegerList class is a template class that provides linked lists. It stores a list of values and provides iterator-based access as well as insertions and removals. IntegerList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

Here's an example of a List that stores integers ...

#define IntegerList IntegerList
#define IntegerListIterator IntegerListIterator
#define integer integer
#define integer integer
#include "List.h90"
#undef IntegerList
#undef IntegerListIterator
#undef integer
#undef integer

{.c}

... and a IntegerList that stores Atom values:

#define IntegerList AtomList
#define IntegerListIterator AtomListIterator
#define integer class(Atom)
#define integer type(Atom)
#include "List.h90"
#undef IntegerList
#undef IntegerListIterator
#undef integer
#undef integer

{.c}

Finally, you can use the first one thus:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
call mylist.append( 8 )
call mylist.append( 5 )
call mylist.append( 1 )
iter => mylist.begin
do while ( associated(iter) )
write(*,*) iter.data
iter => iter.next
end do

{.f90}
