Detailed Description

The StringRealPairList class is a template class that provides linked lists. It stores a list of values and provides iterator-based access as well as insertions and removals. StringRealPairList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

Here's an example of a List that stores integers ...

#define StringRealPairList IntegerList
#define StringRealPairListIterator IntegerListIterator
#define class(StringRealPair) integer
#define type(StringRealPair) integer
#include "List.h90"
#undef StringRealPairList
#undef StringRealPairListIterator
#undef class(StringRealPair)
#undef type(StringRealPair)

{.c}

... and a StringRealPairList that stores Atom values:

#define StringRealPairList AtomList
#define StringRealPairListIterator AtomListIterator
#define class(StringRealPair) class(Atom)
#define type(StringRealPair) type(Atom)
#include "List.h90"
#undef StringRealPairList
#undef StringRealPairListIterator
#undef class(StringRealPair)
#undef type(StringRealPair)

{.c}

Finally, you can use the first one thus:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
call mylist.append( 8 )
call mylist.append( 5 )
call mylist.append( 1 )
iter => mylist.begin
do while ( associated(iter) )
write(*,*) iter.data
iter => iter.next
end do

{.f90}
