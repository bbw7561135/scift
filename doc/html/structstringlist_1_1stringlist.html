Detailed Description

The StringList class is a template class that provides linked lists. It stores a list of values and provides iterator-based access as well as insertions and removals. StringList features both STL-style iterators and Java-style iterators. For developers who already know STL, have the advantage of familiarity.

Here's an example of a List that stores integers ...

#define StringList IntegerList
#define StringListIterator IntegerListIterator
#define class(String) integer
#define type(String) integer
#include "List.h90"
#undef StringList
#undef StringListIterator
#undef class(String)
#undef type(String)

{.c}

... and a StringList that stores Atom values:

#define StringList AtomList
#define StringListIterator AtomListIterator
#define class(String) class(Atom)
#define type(String) type(Atom)
#include "List.h90"
#undef StringList
#undef StringListIterator
#undef class(String)
#undef type(String)

{.c}

Finally, you can use the first one thus:

type(integerlist) :: mylist
type(integerlistiterator), pointer :: iter
call mylist.init()
call mylist.append( 8 )
call mylist.append( 5 )
call mylist.append( 1 )
iter => mylist.begin
do while ( associated(iter) )
write(*,*) iter.data
iter => iter.next
end do

{.f90}
