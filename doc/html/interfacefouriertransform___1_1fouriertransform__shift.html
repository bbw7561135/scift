real(8), dimension(:), optional  oArray

Shift the zero-frequency component to the center of the spectrum.

See Also
http://docs.scipy.org/doc/numpy/reference/generated/numpy.fft.fftshift.html#numpy.fft.fftshift

This function swaps half-spaces for all axes listed (defaults to all). Note that y[0] is the Nyquist component only if len(x) is even.

iArray : array_like. Input array. oArray : array_like. Output array (optional). The shifted array.
