)

Chooses the first one combination with repetitions which satisfy the constraint. This implementation uses a tree-like algorithm with random browsing.

Parameters
[in]nElemsNumber of elements
[in]sGroupssize of the groups
[out]randomItemrandom multiset

example: if S={0,1,3,2} then the combination of these elements in groups of two elements which satisfy the constraint e_i + e_j = 3 are: 1) 0,3 2) 2,1
